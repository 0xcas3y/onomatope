## あ

### ① あたふた
- **意味**：突然のことで慌てて落ち着きを失うさま。「慌てふためく」のくだけた言い方。
- **品詞**：副詞（「〜する」でサ変動詞化可）
- **使い方のコア**：`予期せぬ出来事` ＋ `体を動かして対処しようとしている姿` が揃う場面。心の動揺より **動作の慌て** が前面に出る。
- **よく一緒に使う語**：〜する／〜逃げ出す／〜駆けつける／〜取り出す
- **細かいニュアンスの違い**：

| 語 | 中心にあるもの | 例 |
|---|---|---|
| **あたふた** | 慌ただしい **動作** | 忘れ物に気づいて、あたふた家を出る |
| **おろおろ** | 動揺で **泣きそうな・どうしていいか分からない** 気持ち | 子どもに泣かれておろおろする |
| **うろうろ** | 目的なく **歩き回る** | 駅前をうろうろする |
| **そそくさ** | 慌ただしいが **その場を離れたい** 気持ち | そそくさと席を立つ |
| **ばたばた** | 忙しくて **騒がしい** 動き | 出張準備でばたばたしている |

> ざっくり：**あたふた＝動きが慌てる**／**おろおろ＝気持ちが慌てる**／**うろうろ＝行き先がない**／**そそくさ＝去りたい**／**ばたばた＝忙しい音が出る**。

  1. **HUNTER×HUNTER** 第146話

     > *Ging is actually flustered.*

### ② あっさり
- **意味**：①味・色・性格などがしつこくなく、さっぱりしているさま。②物事が予想外に簡単・手軽に終わるさま。
- **品詞**：副詞／「〜した」で形容詞的に使用
- **使い方のコア**：「濃さ・粘り・抵抗」が **ない** ことを肯定的に言う。味覚・性格・事の運びまで広く使える。
- **よく一緒に使う語**：〜した味／〜認める／〜負ける／〜断る／〜している
- **細かいニュアンスの違い**：

| 語 | 中心にあるもの | 例 |
|---|---|---|
| **あっさり** | **しつこくない／予想より簡単** | あっさり勝負がついた |
| **さっぱり** | **爽快感／きれい・気持ちよい後味** | お風呂でさっぱりした |
| **すっきり** | **もやが晴れた感じ・整理された明快さ** | 説明を聞いてすっきりした |
| **けろっと** | **何事もなかったかのよう** | 叱られてもけろっとしている |

> 「さっぱり」と「あっさり」は味の文脈で混同しやすいが、**さっぱり**は「後味のきれいさ・爽快感」、**あっさり**は「濃くない＝薄味」の意が強い。

## い

### ③ いそいそ
- **意味**：うれしいこと・楽しいことが待っていて、心が弾むように（嬉々として）準備や移動をするさま。
- **品詞**：副詞（「〜と」を伴いやすい）
- **使い方のコア**：**期待に胸を膨らませた外向きの行動**。心の中のワクワクが **動き** にあふれ出ている。自分について使うと気恥ずかしい響きになり、他者を微笑ましく（またはからかって）描写することが多い。
- **よく一緒に使う語**：〜と出かける／〜と支度する／〜とやってくる
- **細かいニュアンスの違い**：

| 語 | 中心にあるもの | 例 |
|---|---|---|
| **いそいそ** | 期待感が **動作** に出ている | デートにいそいそと出かける |
| **うきうき** | 期待感が **気分** にある（動かなくてもよい） | 明日の旅行でうきうきしている |
| **わくわく** | これから起こる事への **胸の高鳴り** | 新作ゲームにわくわくする |
| **にこにこ** | 喜びが **顔（笑顔）** に出ている | プレゼントをもらってにこにこ |

> **いそいそ＝動く**／**うきうき＝気分**／**わくわく＝心臓のドキドキ系の期待**／**にこにこ＝表情**。

### ④ いらいら
- **意味**：物事が思い通りにならなかったり、不快な刺激が続いたりして、神経が逆立つように落ち着かないさま。
- **品詞**：副詞／「〜する」でサ変動詞化
- **使い方のコア**：**とげとげしい不快感**。原因は外（渋滞・騒音・相手の態度）にも内（空腹・眠気）にもある。怒りの **一歩手前の持続的ストレス**。
- **よく一緒に使う語**：〜する／〜が募る／〜をぶつける
- **細かいニュアンスの違い**：

| 語 | 中心にあるもの | 例 |
|---|---|---|
| **いらいら** | 神経が **じりじり** 逆立つ持続的な不快 | 電車が遅れていらいらする |
| **むかむか** | **吐き気的／腹立たしく胃のあたりに来る** 怒り | あの態度、むかむかする |
| **かりかり** | いらいらして **とげとげしい態度を取る** | 何をかりかりしているの |
| **ぴりぴり** | 神経が **過敏で張り詰めた** 緊張状態 | 試験前でぴりぴりしている |
| **うんざり** | 同じ事が続いて **もう懲りた／嫌気** | 毎日の残業にうんざり |

> 怒り系の序列（弱→強）：**うんざり ＜ いらいら ＜ むかむか ＜ かっか／かんかん**。  
> **ぴりぴり**は怒りではなく「ピンと張った緊張」である点が他と違う。
